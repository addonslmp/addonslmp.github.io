# Multiplugin

Multiplugin - это сборка подключаемых плагинов для Lampa.

## Что он делает

- Один раз ставишь - получаешь список всех актуальных плагинов (обновляется из json)
- Включаешь/выключаешь любые плагины галочками
- Есть пункт "только онлайн" - одной кнопкой подгружаются популярные плагины для онлайн просмотра 
- Можно экспортировать выбранные плагины в обычные расширения.
- Пункт "Обновлено" показывает какие плагины были добавлены, а какие удалены.

## Как поставить

1. Lampa → Настройки → Расширения → Добавить плагин
2. Вставляешь эту ссылку:

   https://addonslmp.github.io/multiplugin.js

3. ОК → после добавления плагина, появится новый раздел в настройках - "Мультиплагин"

## Как пользоваться

- Заходишь в раздел "Мультиплагин"
- Нажимаешь на кнопку "Синхронизировать плагины"
- Перезагружаешь Lampa - там есть специальная кнопка для этого)
- Выбираешь категорию и ставишь галочки на нужное.

Если хочешь только онлайн-просмотр без торрентов - жми "Загрузить только онлайн".

## Экспорт

Если плагины понравились - можно экспортировать их в обычные расширения (кнопка "Экспорт включённых").  
После этого они живут отдельно, даже если мультиплагин удалишь.

## Что ещё знать

- Список плагинов лежит тут: https://addonslmp.github.io/sources/plugins_mp.json
- Если что-то сломалось - "Выключить все плагины" + перезагрузка обычно спасает.
- Обновления приходят автоматически при синхронизации.
