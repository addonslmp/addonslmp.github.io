# Multiplugin

Multiplugin is a collection of connectable plugins for Lampa.

## What it does
- Install once and get the list of all current plugins (updated from json)
- Enable/disable any plugins with checkboxes.
- There is an "online only" option - one button loads popular plugins for online viewing.
- You can export selected plugins as regular extensions.
- The "Updated" item shows the update date, which plugins were added and which were removed.

## How to install
1. Lampa → Settings → Extensions → Add plugin
2. Paste this link:
   
   https://addonslmp.github.io/multiplugin.js
   
4. OK → after adding the plugin, a new section will appear in settings - "Multiplugin"

## How to use
- Go to the "Multiplugin" section
- Click the "Sync plugins" button
- Restart Lampa - there is a special button for that
- Choose a category and check the ones you need.

If you want only online viewing without torrents - click "Load online only".

## Export
Plugins can be exported as regular Lampa extensions (button "Export enabled" or "Export → Select plugins").

## Things to know
- Plugin list is here: https://addonslmp.github.io/sources/plugins_mp.json
- If something breaks - "Disable all plugins" + restart usually fixes it.
- Updates come automatically during sync.
