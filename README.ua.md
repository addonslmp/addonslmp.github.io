# Мультиплагін

Мультиплагін - це збірка підключаємих плагінів для Lampa.

## Що він робить
- Один раз встановлюєте і отримуєте список усіх актуальних плагінів (оновлюється з json)
- Вмикаєте/вимикаєте будь-які плагіни галочками.
- Є пункт "Тільки онлайн" - однією кнопкою підвантажуються популярні плагіни для онлайн перегляду.
- Можна експортувати вибрані плагіни в звичайні розширення.
- Пункт "Оновлено" показує дату оновлення, які плагіни додали, а які прибрали.

## Як встановити
1. Lampa → Налаштування → Розширення → Додати плагін
2. Вставляєте це посилання:
   
   https://addonslmp.github.io/multiplugin.js
   
4. OK → після додавання плагіна з’явиться новий розділ у налаштуваннях - "Мультиплагін"

## Як користуватися
- Заходите в розділ "Мультиплагін"
- Натискаєте кнопку "Синхронізувати плагіни"
- Перезавантажуєте Lampa - там є спеціальна кнопка для цього
- Вибираєте категорію і ставите галочки на потрібне.

Якщо хочете тільки онлайн-перегляд без торрентів - жміть "Завантажити тільки онлайн".

## Експорт
Плагіни можна експортувати в звичайні розширення Lampa (кнопка "Експорт увімкнених" або "Експорт → Обрати плагіни").

## Що ще треба знати про плагін
- Список плагінів лежить тут: https://addonslmp.github.io/sources/plugins_mp.json
- Якщо щось зламалося - "Вимкнути всі плагіни" + перезавантаження зазвичай рятує.
- Оновлення приходять автоматично при синхронізації.
